Learn English By Subtitles

Персональный проект для изучения английских слов с помощью субтитров

О проекте

Привет! Я — разработчик проекта Learn English By Subtitles, созданного для изучения английских слов с помощью субтитров. Цель проекта — помочь пользователям быстро и эффективно расширять словарный запас и отслеживать прогресс.

Что умеет проект

  • Прохождение нескольких уровней тестирования для закрепления изученных слов.
  • Создание персональных списков слов для удобного изучения.
  • Получение статистики изученных слов и отслеживание прогресса.

Все функции разработаны с упором на простоту и удобство пользователя.

Мои навыки и вклад

Проект полностью разработан мной самостоятельно:

  • Бэкенд: Python, Django, Django REST Framework, Celery — обработка данных, логика тестов и асинхронные задачи.
  • Фронтенд: HTML, CSS, JavaScript, Bootstrap 5 — адаптивный интерфейс и динамическое обновление данных через AJAX.
  • Работа с данными: SQLite, PostgreSQL/MySQL, SQLAlchemy — хранение и обработка списков слов и субтитров.
  • Инструменты разработки: Git/GitHub для контроля версий, Docker для локального тестирования и развертывания проекта.

Я самостоятельно продумывал архитектуру сайта, структуру базы данных и взаимодействие всех компонентов, что позволило закрепить навыки полноценной разработки веб-приложений.

Презентация проекта

Learn English By Subtitles демонстрирует мою способность создавать функциональные и удобные веб-приложения, работать с базой данных, обрабатывать файлы и строить динамический интерфейс. Проект отражает мою внимательность к деталям, умение планировать и реализовывать продукт от идеи до готового результата.
